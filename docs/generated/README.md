# docs/generated/

This directory is managed by the **Gen AI Auto-Doc pipeline** (Track A — developer reference docs).

Auto-generated developer reference documentation is committed here via
a GitHub Draft PR after each code change. Do not edit files in this
directory manually — they will be overwritten on the next pipeline run.

## Contents

| File | Description |
|---|---|
| `auth_service_reference.md` | Reference docs for `auth_service/auth.py` |
| `user_reference.md` | Reference docs for `auth_service/user.py` |
| `frontend_client_reference.md` | Reference docs for `frontend-client/apiClient.ts` and `frontend-client/authHooks.ts` |

## File Naming Convention

Each generated file follows the pattern `<module_name>_reference.md`, one file per documented source module.
Files are created or overwritten by the DocumentationAgent when the corresponding source file changes.
