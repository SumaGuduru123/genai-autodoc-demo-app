### `divide(a: float, b: float) -> float`

Returns `a` divided by `b`. Raises `ValueError` if `b` is zero.

| Parameter | Type | Description |
|---|---|---|
| `a` | `float` | Dividend |
| `b` | `float` | Divisor — must not be zero |

**Returns:** `float` — the arithmetic quotient.

**Raises:**
- `ValueError` — when `b == 0` with message `"Cannot divide by zero."`

**Usage example:**
```python
result = divide(10, 2)   # → 5.0
result = divide(7, 2)    # → 3.5

# Error case:
try:
    divide(5, 0)
except ValueError as e:
    print(e)  # Cannot divide by zero.
```
